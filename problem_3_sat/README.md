Walk Sat
=============

Implements a Metropolis Hastings type Monte Carlo algorithm solving the Max 3-SAT Problem

## Step 1
Compile Code
```
g++ WalkSat.cpp -o WalkSat
```

## Step 2
Run Code
```
./exec.sh <input_file> out.tsp
```

## With Timing
```
./run_program.sh ./exec.sh <input_file> out.tsp
```
